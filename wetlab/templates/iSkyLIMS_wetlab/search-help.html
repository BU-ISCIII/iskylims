Searching for a Run

To look for a specific run you should enter as much as possible details you have, in order to find an exact match
In case that more than one result are matched then, a list is displayed.

  • On RUN level
    • Looking for an especific run: Filling the complete or partial run name
    • Looking for an especific Platform: Selecting the available platfrom from the list
    • Looking for State RUN : Selecting a run state from the list
    • Searching from an especific date: Filling the Date Field
    • Searching for run State: Selecting from the state list Field

    • On Project level
      • Looking for an especific project: Filling the complete or partial project name
      • Looking for an especific Platform: Selecting the available platfrom from the list
      • Looking for State RUN : Selecting a run state from the list
      • Searching from an especific date: Filling the Date Field
      • Searching for Project State: Selecting from the state list Field

      • Searching for project responsible user : Wetlab Manager can search by any investigator.
