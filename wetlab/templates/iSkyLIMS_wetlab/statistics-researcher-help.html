Statistics done on a researcher

This page will provide the statistics based on the researcher name.

The researcher name field is a mandatory field and it cannot be blank.
When filling the addtional fields in the form, the number of matches will be reduced to the ones that fullfil all conditions defined on the fields

As a result after sending the request you could get the following:

  • Error page, if there is no match in the request
  • Page containing all information related to project done by the researcher
  • Page containing a list of the project that matches with the defined conditions

Read the foolowing tips for searching proyects done by researcher

  • Looking for projects done by the researcher: Filling the exact name of the researcher name field
  • Looking for the latest project which matches all conditions: Clicking on the Latest project checkbox
  • Searching from an especific date: Filling the Date Field
  • Searching for project State: Selecting from the state list Field
